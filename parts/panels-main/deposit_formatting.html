Other Formatting Tools

RCSB Data Extraction and Deposition Preparation Tools

  • pdb_extract – Tools and examples for extracting mmCIF data from structure determination applications (workstation/command-line version)
  • sf-convert  – A format conversion tool for structure factor files that supports more than a dozen file formats

RCSB Data Format Conversion Tools

  • MAXIT  - An application for processing and annotating of macromolecular structure data (Updated! March 1, 2013)
  • PDBML2CIF  - A tool for converting PDBML data files into mmCIF data files (Updated! November 18, 2013)
  • PointSuite - generate BIOMT and CIF symmetry records for macromolecular assemblies with point and helical symmetries (Updated! May 31, 2012)
